PDFTools Pro - Deployment Package
=====================================

This package contains your professional PDF tools web application ready for deployment on Hostinger.

IMPORTANT: Read the deployment-guide.md file for detailed step-by-step instructions.

Quick Start:
1. Check if your Hostinger plan supports Node.js hosting
2. Upload all files to your public_html directory
3. Install dependencies: npm install
4. Start the application: npm start

Files included:
- index.js (main server)
- package.json (dependencies)
- public/ (frontend files)
- temp/ (upload directory)
- output/ (processed files directory)
- deployment-guide.md (detailed instructions)

For questions or support, refer to the deployment guide.
